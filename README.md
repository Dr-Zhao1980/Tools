## � 项目简介

本项目是一个个人工具箱仓库，收录了多年来收集和使用的各类实用工具。所有工具都经过实际使用验证，确保其可用性和实用性。工具涵盖了系统管理、网络诊断、开发辅助等多个领域，适合日常使用和故障排除。

## 📁 工具分类

### 🌐 网络工具

#### 360网络急救箱
- **版本**: 集成版 + 完整版
- **描述**: 360公司出品的网络诊断和修复工具
- **主要功能**:
  - 网络连接诊断与故障排查
  - 网络配置自动修复
  - 网络流量实时监控
  - 系统网络参数优化
  - DNS设置修复
  - 网络适配器问题检测
- **适用场景**: 网络无法连接、网速慢、DNS解析失败等问题
- **位置**: `360网络急救箱/`
- **使用说明**: 以管理员权限运行主程序，按照向导操作

### 💾 磁盘管理工具

#### SpaceSniffer 汉化版
- **版本**: 1.1.2.0 汉化版
- **描述**: 可视化磁盘空间分析工具
- **主要功能**:
  - TreeMap可视化布局显示文件分布
  - 快速扫描和分析磁盘占用
  - 智能缓存扫描引擎，扫描速度快50%
  - 支持实时浏览和缩放
  - 强大的过滤功能
  - 与系统事件同步
  - Unicode完整支持
- **特色**: 直接汉化版，单文件独立运行，无需语言包
- **位置**: `SpaceSniffer汉化版/`
- **使用说明**: 直接运行 `SpaceSniffer.exe`，选择要分析的磁盘

### 🛡️ 系统管理工具

#### Autoruns 汉化版
- **版本**: 14.11 64位单文件版
- **描述**: Windows系统启动项管理工具
- **主要功能**:
  - 管理系统启动程序
  - 查看和控制系统服务
  - 管理驱动程序加载项
  - 计划任务管理
  - 浏览器插件管理
  - 系统钩子检测
- **特色**: Sysinternals官方工具汉化版，功能强大
- **位置**: `Autoruns汉化/`
- **使用说明**: 以管理员权限运行，可查看和管理所有启动项

#### 禁用Windows Defender工具
- **版本**: 1.1
- **描述**: Windows Defender一键禁用和卸载工具
- **主要功能**:
  - 一键禁用Windows Defender实时保护
  - 完全卸载Windows Defender组件
  - 恢复系统默认设置
- **适用场景**: 需要安装其他杀毒软件或解决兼容性问题
- **位置**: `禁用WindowsDefinder/`
- **使用说明**: 以管理员权限运行，选择相应操作

## 🚀 使用指南

### 通用使用步骤
1. **选择工具**: 根据需求选择相应的工具目录
2. **权限准备**: 大部分工具需要管理员权限
3. **运行程序**: 双击运行对应的可执行文件
4. **按提示操作**: 根据工具界面提示进行相应操作

### ⚠️ 重要注意事项

#### 安全提醒
- **管理员权限**: 系统修改类工具必须以管理员身份运行
- **杀毒软件**: 某些工具可能被杀毒软件误报，请添加信任
- **系统兼容性**: 确认工具与您的Windows版本兼容
- **数据备份**: 使用系统修改工具前务必备份重要数据

#### 使用建议
- **虚拟机测试**: 建议先在虚拟机中测试工具功能
- **按需使用**: 仅在确实需要时使用系统修改工具
- **定期更新**: 关注工具版本更新，获取最新功能
