这是一款非常适合喜欢折腾、对电脑技术感兴趣的同学使用的神器。它的界面极其精简，但为了确保你的数据安全，我们需要非常严谨地按照步骤来操作。

下面是 **FreeMove** 最详尽的使用图文级拆解教程，我们以“把 C 盘的微信迁移到 D 盘”为例：

### 🛑 零步：极其重要的准备工作

在你开始点击任何按钮之前，**必须确保你要迁移的软件已经彻底关闭**。

* 如果软件正在运行（比如微信还在后台挂着），系统会锁死相关文件，导致迁移失败甚至文件损坏。
* **检查方法**：按 `Ctrl + Shift + Esc` 打开任务管理器，确保你要搬家的软件进程已经完全结束。

---

### 第一步：获取软件并获取“最高权限”

1. **下载**：从 GitHub 下载 FreeMove 的 `.exe` 可执行文件。它是一个绿色软件，不需要安装。
2. **获取系统权限（核心步骤）**：因为创建“目录联接”需要修改 Windows 的底层目录结构，普通的双击运行是会报错的。
* **操作**：找到下载好的 `FreeMove.exe`，**右键点击它**，在弹出的菜单中选择 **“以管理员身份运行”**。



---

### 第二步：认识极简界面与配置路径

打开软件后，你会看到一个非常简单的窗口，主要有两个输入框和一个复选框：

1. **Move From（要搬走的原地址）**
* 这是你要迁移的软件**当前所在的 C 盘文件夹**。
* 点击右侧的 `Browse...`（浏览）按钮。
* 找到你要搬家的文件夹。比如微信默认在：`C:\Program Files (x86)\Tencent\WeChat`。选中 `WeChat` 这个文件夹。
* *注意：一定要选到软件自身的根目录，不要只选上一级的 Tencent 文件夹。*


2. **To（搬去的新家）**
* 这是你想把软件**存放在哪个盘**。
* 同样点击右侧的 `Browse...` 按钮。
* 比如你在 D 盘建了一个名为 `D:\SoftwareMoved` 的文件夹。你就选中它。
* *FreeMove 会自动在 D 盘这个位置创建一个同名文件夹（即复制过去的 WeChat）。*


3. **Set original folder to hidden（将原文件夹设为隐藏）**
* 这是一个**复选框**（打勾的地方）。
* **强烈建议打勾**。它的作用是：当软件被成功转移，并在 C 盘留下那个“隐形传送门”（带有快捷方式箭头的同名文件夹）时，系统会自动把它隐藏起来。这样你的 C 盘看起来会更清爽，也能防止你以后手滑误删了这个关键的“传送门”。



---

### 第三步：一键迁移 (Move)

确认路径无误后：

1. 点击右下角的大按钮 **`Move`**。
2. 接下来软件会进入无响应状态或者弹出进度条（取决于文件有多大）。此时**千万不要强行关闭它**。
3. 它在后台正在做三件事：
* 把几十上百 GB 的文件原封不动复制到 D 盘。
* 删除 C 盘的原文件。
* 敲下系统命令，在 C 盘建立关联到 D 盘的“目录联接”。


4. 当弹出 `Done` 或者 `Success` 的提示框时，恭喜你，迁移完成！

---

### 第四步：验收成果与底层探秘

作为喜欢探究底层逻辑的玩家，你可以去 C 盘原来的位置看看发生了什么：

1. 打开 C 盘你刚才迁移的路径（如果你刚才勾选了隐藏，你需要在文件资源管理器的“查看”菜单里，勾选“隐藏的项目”才能看到）。
2. 你会发现原来那个 `WeChat` 文件夹还在，但是它的图标左下角**多了一个类似于快捷方式的小箭头**。
3. 双击进入这个文件夹，你会发现里面的文件全都在。但如果你查看 C 盘的剩余空间，会发现容量已经腾出来了！你在这里看到的“文件”，实际上只是 D 盘文件的“倒影”。
4. 现在，正常双击桌面上的微信快捷方式，你会发现它依然能完美秒开，软件根本不知道自己已经被“偷天换日”了。

---

### ⚠️ 高阶必修：后悔药（如何还原？）

如果在未来的某一天，你想把它恢复原样（比如 D 盘要格式化了），你应该怎么做？这也是玩底层逻辑必须掌握的“擦屁股”技能：

**千万不要用 FreeMove 往回移！** 手动还原是最安全的：

1. 同样，确保该软件已经彻底关闭。
2. 进入 C 盘，找到那个带有小箭头的“传送门”文件夹，**直接把它删除**（放心，删除联接点绝对不会删除 D 盘里的真实文件）。
3. 去 D 盘，把那个真实的文件夹，**手动“剪切”并“粘贴”**回 C 盘原来的位置。
4. 一切就恢复成最初没动过的样子了。

理解了这套“目录联接”的逻辑，以后无论是应对 C 盘爆满，还是处理某些强制只能安装在系统盘的流氓软件，你都能轻松拿捏了！要不要先拿个无关紧要的小软件测试一下手感？
